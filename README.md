## Important commands

 - Transpile to .js on file change
```
    $ yarn watch
```

 - Watch for .js changes
```
    $ yarn dev
```

 - Generate new migration
```
    $ yarn typeorm:generate -n "MigrationName"
```

 - Run all migrations
```
    $ yarn typeorm:run
```
